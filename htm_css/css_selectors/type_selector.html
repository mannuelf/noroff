Type Selectors

The type selector allows you to style HTML element nodes by the node name.

. . . 
  h1 {
      font-family: Hack, sans-serif, Helvettica;
      color: #000000;
    }

  span {
    font-size: 4em;
    color: #000000;
  }

  div {
    display: flex;
    border: 1p solid #FF9900;
  }
